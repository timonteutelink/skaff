title: Getting Started

Welcome aboard!  This guide walks you from **nothing** ➜ **first generated project** in 5 minutes.

## 1. Install

```bash
bun add -g @timonteutelink/code-templator-cli
````

> **Bun required ≥ 1.2** – use `bun upgrade` if needed.

## 2. Generate something

```bash
code-templator new next-api my-api
```

The CLI scaffolds a typed Next.js API route:

* ✅ ESLint, Prettier, testing
* ✅ Environment-driven config
* ✅ Ready for Vercel serverless

## 3. Explore the files

| File                       | Purpose                        |
| -------------------------- | ------------------------------ |
| `api/hello.ts`             | Example endpoint *Hello World* |
| `package.json`             | Scripts for linting & deploy   |
| `.github/workflows/ci.yml` | Pre-set CI pipeline            |

## 4. What’s next?

* Dive into the **Library API** to script your own generator.
* Or head over to **Web Architecture** for a deep dive into the Next.js UI.
