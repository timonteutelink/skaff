## Quick taste

<CodeBlock language="bash">
npx code-templator new react-app my-todo
cd my-todo
bun dev
</CodeBlock>

This spawns a fully-wired React + Bun dev server in **3 seconds**.

---

## Why another templator?

* **Typed** – written in TypeScript end-to-end  
* **Composable** – expose primitives so you can build your own generators  
* **Zero lock-in** – it only writes files; no runtime dependency
